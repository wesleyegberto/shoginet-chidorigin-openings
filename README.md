# Chidorigin Shogi Openings Translation

[Chidorigin's Opening Encyclopedia](http://shogi-chess.net/senpouzukan/) translation using LLM.

What was done:

- the website was scraped to download the pages and kifu
- the link for the static resources from original site is used
- the original pages was kept under the name of `raw.html`
- the translation was done using LLM, some mistake was fixed during my reading
